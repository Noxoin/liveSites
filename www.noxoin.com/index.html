Chinese Music Tracking

An application to track chinese music I've come across on YouTube.

This contains 2 portions of the application; a chrome extension and a web application. The chrome extension will send a REST call to the api if it detects a chinese song from JoyRich. This in turn will allow me to manage new songs listened to, star the favourites and use the application as a music player.
